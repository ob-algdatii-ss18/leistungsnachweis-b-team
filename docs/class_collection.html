Detailed Description

template<class T>
class Collection< T >

Template Klasse für alle Listen die verglichen werden sollen.

Diese Klasse enthält die Methodenvorgaben und Mechanismen für die Vergleichsmessung. Dazu kann beim Einfügen, Suchen und Entfernen die Anzahl der Vergleichsoperationen und Dateizugriff erfasst werden.
